Project Title
Smiling Emoji Using HTML & CSS

Project Overview

This project demonstrates how to create a simple and attractive smiling emoji using only HTML and CSS. The emoji is designed using basic HTML elements and styled with CSS properties such as border-radius, positioning, colors, and transforms.

Features

* Pure HTML and CSS implementation
* No images or external libraries required
* Responsive and lightweight design
* Demonstrates CSS positioning and styling techniques
* Beginner-friendly project for learning web design

Technologies Used

* HTML5
* CSS3

Project Structure

* smiling.html – Contains the structure of the emoji.
* style.css – Contains all styling and design elements.

Refer to smile.png

Learning Outcomes

Through this project, I learned:

* Creating shapes using CSS
* Using border-radius to design circular elements
* Positioning elements with relative and absolute positioning
* Styling web pages with colors and transformations
* Building creative UI components without images

How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open smiling.html in any modern web browser.
4. The smiling emoji will be displayed on the screen.
